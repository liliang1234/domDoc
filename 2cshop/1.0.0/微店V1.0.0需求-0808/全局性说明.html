全局性功能说明

1、在手机浏览器打开微店时，所有可点击的相机icon、扫码icon全都屏蔽；

2、在微信中打开微店时，所有可点击的相机icon、扫码icon显示（前提是，微信调用相机的授权接口可用，若不可用，则屏蔽这些icon和按钮）

3、微信中打开时，所有页面的头部名称和左侧回退icon全部隐藏（注意：通用产品组详情页右上角的二维码icon不隐藏）

4、其他浏览器打开时，所有页面的头部名称和左侧回退icon全部显示
