一、说明

影响通用和票产品下单页选择游玩日期的字段为：产品可用日期、关闭时间

 

二、月份展示规则

1、当商品可用日期的开始日期早于当前日期，商品可用日期的结束日期晚于当前日期，展示月份为从当前日期的月份开始的三个月，比如：商品可用日期为6月1日-10月1日，当前日期为7月22，展示月份应为 7月、8月、9月。

2、当商品可用日期的结束日期早于当前日期，展示月份为从当前日期的月份开始的三个月。

3、当商品可用日期的开始日期晚于当前日期，展示月份为商品可用日期-开始日期的月份开始的三个月。

比如：商品可用日期为9月1日-12月1日，当前日期为7月22日，展示月份应为 9月、10月、11月。

 

三、游玩日期日历展示具体规则

1、展示可选中的日期为：产品可用日期（去掉已过去的日期）减去关闭时间。如，产品可用日期为2016年7月1日至7月31日，当前日期为7月22日，关闭时间为7月22、23、24日，则游玩日期日历中展示可选中的日期为：7月25、26、27、28、29、30、31日（不可选的日期要置灰，不可点击）
