3.搜索结果排序逻辑

（1）搜索结果匹配：

先精准匹配景区或通用产品组名称，再模糊检索

 

（2）排序规则

精准匹配结果优先于模糊检索结果

 

B 搜索无结果页 (具体页面以ui设计为准）

 搜索无结果时，提示用户

抱歉，暂没搜到相关景区！

您可更换下关键词再试试

（3） 搜索记录

A展示条件：单击输入框输入关键词时，展示搜索下拉框，具体布局见原型

B 用户无搜索记录：搜索记录部分展示文案“暂无搜索记录”并展示热门搜索词(具体样式以ui设计为准）

C用户有搜索记录： 按用户搜索的关键词的时间倒叙排序，默认只展示5条。超出部分进行收起，由用户自助点开后在全部展示 

D 热门搜索：后台增加热门搜索控制模块，人为配置热搜词
